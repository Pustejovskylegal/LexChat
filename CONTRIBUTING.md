# Přispívání do projektu

Děkujeme za váš zájem o přispívání do LexChat!

## Jak přispět

1. Forkněte repozitář
2. Vytvořte feature branch (`git checkout -b feature/AmazingFeature`)
3. Commitněte změny (`git commit -m 'Add some AmazingFeature'`)
4. Pushněte do branch (`git push origin feature/AmazingFeature`)
5. Otevřete Pull Request

## Coding Standards

- Používejte TypeScript pro všechny nové soubory
- Dodržujte existující code style
- Přidejte komentáře k složitějším částem kódu
- Testujte změny před commitnutím

## Commit Messages

Používejte jasné a popisné commit messages:
- `feat:` pro nové funkce
- `fix:` pro opravy bugů
- `docs:` pro změny v dokumentaci
- `style:` pro formátování
- `refactor:` pro refaktoring
- `test:` pro testy

## Issues

Před vytvořením Pull Requestu prosím:
- Zkontrolujte existující issues
- Vytvořte issue pro větší změny
- Popište změny v Pull Requestu

## Bezpečnost

- **Nikdy** necommitněte API klíče nebo citlivé informace
- Všechny secrets musí být v `.env` souboru
- `.env` soubor je již v `.gitignore`

Děkujeme za váš příspěvek!
